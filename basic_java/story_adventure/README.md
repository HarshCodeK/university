# Interactive Story Adventure Game

A terminal-based "Choose Your Own Adventure" game with 20+ decision points.

## Features

- Multiple branching story paths
- Tracks user choices
- Displays total possible story combinations at the end
- ASCII art ending screen

## How to Compile and Run

```bash
# Compile
javac StoryGame.java

# Run
java StoryGame
```

## Story Overview

You wake up in a jungle with no memory and discover an ancient temple. Based on your choices, you may find treasure, gain wisdom, or meet an untimely end!
