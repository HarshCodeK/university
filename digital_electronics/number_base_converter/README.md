# Universal Number Base Converter

A modern, visually appealing web tool for converting numbers between Binary, Decimal, Hexadecimal, and Octal.

## Features

- Auto-detect input base or manual selection
- Real-time conversion between all bases
- Step-by-step animated explanation for each conversion
- Clean, modern UI with dark/light mode toggle
- Responsive design

## How to Use

Simply open `index.html` in any modern browser.

1. Type a number in the input field
2. Select the input base (or use "Auto Detect")
3. Click any result card to see step-by-step explanation
